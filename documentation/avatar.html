Avatar

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in any web application.

Example of circular avatars

Avatar is available in 5 different sizes.Classes like 'avatar-xsmall', 'avatar-small', 'avatar-medium', 'avatar-large', 'avatar-xlarge' can be applied to change the size of avatar.

Images on avatar can be changes by replacing the source of image in the code.

Example of rounded corner avatars

Avatar's shape can be changed to rounded corners by applying the 'rounded-corners' class on it.

Example of square avatars

Avatar's shape can be changed to square by applying the 'square' class on it.
