Images

Images can be responsive to fit the parent's width, and also can be customised to be round shaped.

Example of Circular Image

Image can be made responsive by using class 'imge-resp'.By using class 'image-round' shape of image becomes circular/oval depending on height of image.

Example of Image with rounded corners.

Image can be made responsive by using class 'imge-resp'.By using class 'image-round-corners' corners of image become soft/rounded.
